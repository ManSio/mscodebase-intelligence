### Что значат цифры

**Margin** — насколько релевантный документ ближе к запросу, чем лучший нерелевантный:
- `0.523` = Очень хорошее разделение (Qwen3)
- `0.348` = Хорошее разделение (BGE-M3)
- `0.182` = Слабое разделение (Granite)

**Все модели показали 100% Accuracy (rank=1)** — правильный документ всегда был самым похожим.

## 🎯 ФИНАЛЬНАЯ РЕКОМЕНДАЦИЯ

### Выбор провайдера (автоматический)

```
Система >= 16 GB RAM → Qwen3-Embedding-0.6B (максимальное качество)
Система 8-16 GB RAM  → BGE-M3 (оптимальный баланс)
Система < 8 GB RAM   → BGE-M3 (единственный viable вариант)
```

### Рекомендуемая конфигурация (сейчас)

```yaml
embedder:
  primary: bge-m3              # 692 MB, отличное качество для кода+русского
  secondary: qwen3-embedding   # 1669 MB, для пользователей с 16+ GB RAM
  fallback: onnx-server        # 757 MB, стабильный fallback

reranker:
  primary: bge-reranker-v2-m3  # Единственный viable мультиязычный reranker
  future: qwen3-reranker       # Когда появится GGUF
```
