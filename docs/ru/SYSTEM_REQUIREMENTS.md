# Системные требования и справочник по архитектуре

> Основано на реальных тестах (2026-07-10) на Windows 11 Insider Preview сборка 26220
> CPU: AMD Ryzen 5 5600H (12 ядер) | RAM: 16 ГБ DDR4

---

## 1. Минимальные требования

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| **CPU** | x86_64 с AVX2 (AMD Ryzen 3xxx / Intel Haswell 4xxx+) | Любой x86_64 с AVX2 |
| **RAM** | 8 ГБ (система) + 1.2 ГБ (MCP + модели) | 16 ГБ + 1.2 ГБ |
| **Диск** | 2 ГБ свободно (модели + индекс) | 5 ГБ для больших монорепозиториев |
| **ОС (Windows)** | Windows 10 22H2+ | Windows 11 stable (не Insider Preview) |
| **ОС (macOS)** | macOS 12+ Intel или Apple Silicon | macOS 14+ Apple Silicon (Metal) |
| **ОС (Linux)** | Ubuntu 22.04+ x86_64 | Ubuntu 24.04+ |
| **Python** | 3.10+ | 3.12+ |
| **Zed** | v1.10.0+ | последняя |

## 2. Разбивка RAM (реальные измерения)

### Текущая архитектура (Qwen3 + BGE-M3 через llama.cpp GGUF)

| Процесс | RAM (без --mlock) | RAM (с --mlock) | Примечания |
|---------|-------------------|-----------------|------------|
| **MCP (python.exe)** | 213 – 252 МБ | 213 – 252 МБ | Стабильно, без утечек |
| **llama-server Qwen3** (порт 8080) | **346 МБ** | 772 МБ | Модель эмбеддингов |
| **llama-server BGE-M3** (порт 8081) | **450 МБ** | 539 МБ | Модель реранкера |
| **Всего** | **~1.0 – 1.1 ГБ** | ~1.5 – 1.6 ГБ | |

### Сравнение по датам

| Дата | RAM MCP | RAM Моделей | Всего | Конфигурация |
|------|---------|-------------|-------|--------------|
| 2026-07-05 | 185 МБ | 0 МБ (внешний API) | **185 МБ** | LM Studio (удалённо) |
| 2026-07-08 | 172 МБ | 0 МБ (внешний API) | **172 МБ** | LM Studio |
| 2026-07-09 (раннее) | 151 МБ | 0 МБ (внешний API) | **151 МБ** | LM Studio недоступен |
| 2026-07-09 (позднее) | 1,931 МБ | 757 МБ | **2,688 МБ** | ONNX в процессе 🔴 |
| **2026-07-10** | **252 МБ** | **796 МБ** | **~1,048 МБ** | **Qwen3 + BGE-M3 GGUF** 🟢 |

**Ключевой вывод:** ONNX Runtime в процессе MCP вызывал 2 ГБ+ RAM. Переход на llama.cpp GGUF снизил память MCP на 88% и общую на 60%.
